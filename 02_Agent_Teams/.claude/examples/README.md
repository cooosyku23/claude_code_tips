# エージェントチーム 出力例

このディレクトリには、`/agent-team` コマンドが生成するプロンプトの参考例を配置します。

メタプロンプトはプロンプト生成時にこのディレクトリを参照し、出力スタイルや構成の参考にします。

## 使い方

1. 良いプロンプト例ができたら、このディレクトリに `.md` ファイルとして保存する
2. ファイル名はタスクの種類がわかるように命名する（例：`research-auth-best-practices.md`）
3. 例が増えるほど、メタプロンプトの出力品質が安定する

## 現在の例

- `research-auth-best-practices.md` — 調査・リサーチ型の例（Node.js 認証ベストプラクティス）
